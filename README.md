# SDD Dual Loop

一个以“意图环 + 执行环”为核心的软件开发工作流实验项目。

当前里程碑是 **M1：双环最小可运行闭环**。M1 仅实现单仓库、单模型、顺序执行的小型 FastAPI 功能开发路径；完整边界见 [`docs/milestones/M1.md`](docs/milestones/M1.md)。

架构总纲见 [`docs/dual-loop-architecture.md`](docs/dual-loop-architecture.md)。该文件是项目“宪法”，MVP 实施期间不随意修改。

## 当前状态

完整目录已建立，业务模块暂只标明职责。后续实现以 M1 最短路径为限，不提前建设并行编排、模型路由、注册中心或企业治理能力。
